# Netflix Content Analysis

Data visualization and exploratory data analysis project using the Netflix dataset.

## Project Overview

This project analyzes Netflix content trends, including:

- Movies vs TV Shows
- Content growth over time
- Genre distribution
- Country-based analysis
- Rating analysis
- Netflix release trends

The project focuses on extracting insights from Netflix data through data cleaning, visualization, and exploratory analysis.

---

## Dataset

The dataset contains information about Netflix titles, including:

- Title
- Type
- Country
- Release year
- Date added
- Rating
- Duration
- Genres
